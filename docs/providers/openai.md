# OpenAI

Set in `scout.config.json`: `"model": "gpt-4o"` (any `gpt…`, `o1`, `o3`, or `o4…` id). Runs on the **AI SDK** engine.

## Credentials

1. **`OPENAI_API_KEY`** — the only source.

## Setup

```bash
# get a key at https://platform.openai.com/api-keys
export OPENAI_API_KEY=sk-...
scout doctor
```
